# VoteTalk
VoteTalk은 간단하고 사용하기 쉬운 온라인 투표 플랫폼입니다.

## 주요 기능
- 투표 생성 및 관리
- 실시간 투표 결과 확인
- 댓글 기능을 통한 토론
- 관리자 기능

## 기술 스택
- Backend: FastAPI
- Database: SQLite + SQLAlchemy
- Frontend: Jinja2 Templates
- Authentication: JWT

## 설치 및 실행
### Docker Compose 실행
1. 저장소 클론
```bash
git clone https://github.com/yourusername/votetalk.git
cd votetalk
```
2. Docker Compose 실행
```bash
docker-compose up -d
```
